# Carpeta de imágenes

Coloca aquí los recursos gráficos del sitio (logotipos, fotografías, ilustraciones).

Formatos recomendados:
- **SVG** para iconos y logotipos.
- **WebP / JPG** optimizados para fotografías.
- **PNG** cuando necesites transparencia.

> `favicon.svg` se usa como ícono del sitio.
